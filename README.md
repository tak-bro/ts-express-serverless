# ts-express-serverless

## How to use

### Installation

```sh
$ npm install
```

### Build for typescript

```sh
$ npm run build
```

### Development

```sh
$ npm start # run express
$ npm run start.serverless # run as serverless.offline
```

### Test

```sh
$ npm test
$ npm run dev:test # for development
```

### Lint

```sh
$ npm run lint
```

### Deploy to AWS

```sh
$ npm run deploy
```
